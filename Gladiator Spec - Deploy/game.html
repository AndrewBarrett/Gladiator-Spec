Quick Start & Known Bugs

Playing the Sim

To get started wait for the sim to load, hit the menu option "Gladiator Spec Sim" and you will be presented with the main interface. Click on a target dummy which will highlight it and hit charge (default keybind: Z) - this will begin auto attacks and start filling up the rage bar allowing you to use abilities.

For an explanation of the mechanics of each ability take a look at the guide which is linked at the top of the page.

Keybinds

The default keybinds have been setup to be the same as what I use in the live game. I've been told many times by my guildies that my keybinds are fucked up and I suspect that there's a high chance you might agree. If you go into the settings section from the main menu most of it hasn't been built yet but the keybinding section fortunately has.

To change a keybind click the relevant button you want to rebind, mouse over the "KEYBIND" frame so it is highlighted, hit the keybind you want to change it to (A-Z, 0-9), and then hit the "Set Keybind" button. Note that there is support for Modifiers (CTRL, SHIFT, ALT) but they will not function when the sim is being played through a browser.

Bugs

It's been a long time since I've coded anything so please accept my apologies for some pretty rusty code. There's a few performance issues particularly when lots of damage events are on the screen. The most irritating one I've found so far is that scrolling down the page will cause the cursor to not align properly with the sim window making clicking on menu objects a pain in the ass.

If you find a particularly crappy bug feel free to message me and it will get added to the list.
