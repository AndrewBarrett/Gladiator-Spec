Gladiator Spec Re-Design

By Garrow

The beauty of Gladiator Stance in WoD was that it only took one talent at the end of the Protection tree to single handedly transform an entire tanking spec into a (in my opinion) fun and engaging dps spec. The inherent limitation that came with it is that Gladiator Stance was not an individual spec in its own right and going forward into Legion (especially with artifact weapons) would have posed a number of development problems. I would personally love to see Gladiator Stance come back to the game as a spec in its own right and for a bit of fun I came up with and coded a re-design of the spec that should be playable in most browsers.

With this redesign I've attempted to keep a similar feel to how Gladiator Stance played during WoD while hopefully giving it a bit more of it's own identity to seperate it from the other specs. While I would never expect something like this to actually make it into the live game it's still been a bit of fun to make and you can find a link giving a brief explanation on how to play the spec at the top of the page. For anyone curious there's also a link to the source code on Git Hub below.

Right now talents haven't been properly implemented but they should be enabled shortly.
